# public/ — Frontend (served by Cloudflare Pages)

Static files deployed via `npm run deploy:ui` (wrangler pages deploy). API calls go to tower-nas backend through Cloudflare Tunnel.

| File | Purpose |
|------|---------|
| **index.html** | Main report dashboard. Alpine.js SPA. Shows site cards (sidebar), paginated PDF doc list with scores/thumbnails, upgrade queue, cost tracking. Polls `/api/activity` every 3s for pipeline progress. Admin mode unlocks: upgrade, reset, focus buttons. |
| **receipt.html** | PDF upgrade receipt. Rendered as a perforated tear-off card. Shows per-stage cost, quality delta, OCR confidence, narrative. Loaded in an iframe modal from index.html. |
| **viewer.html** | Full-page PDF viewer via pdfjs. Zoom, page nav, text layer, search. |
| **sw.js** | Service worker. Caches only static assets (CSS/JS/images) — never HTML. HTML always network-fetched so deploys are immediately visible. Cache key: `site2rag-__BUILD_TIME__` (replaced at deploy time by deploy-ui.js). |
| **manifest.json** | PWA manifest. |
| **version.json** | `{v, build}` — bumped by scripts/bump-version.js on each deploy. |
| **tailwind.css** | Generated by `npm run build:css`. Do not edit by hand. |
